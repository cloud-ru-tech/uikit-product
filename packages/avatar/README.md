# React Avatar

## Installation
`npm i @sbercloud/uikit-react-avatar`

React component for drawing avatar component or placeholder

## Components interface 
```typescript
type AvatarProps = {
  name?: string;
  variant: Variants;
  size?: Sizes;
  src?: string;
  status?: Status;
  onClick?: MouseEventHandler;
  className?: string;
};
```
