#Contribution guide

Для того, чтобы добавить новую иконку - просто поместите еще в одну из трех папок:
- `svg`
- `svg-color`
- `svg-services`

## NPM Scripts
* `build:icons` - сборка папки svg
* `build:services-icons` сборка папки svg-color
* `build:icons-color` сборка папки svg-services
* `compile` - вызывает все три, НЕ ПЕРЕИМЕНОВЫВАТЬ, вызывается при билде всей монорепы
