# React Divider

## Installation
`npm i @sbercloud/uikit-react-divider`

## Props 
```typescript
type DividerProps = {
  className?: string;
  variant?: Variant;
  orientation?: Orientation;
};

enum Variant {
  Primary = 'primary',
  Secondary = 'secondary',
  Accent = 'accent',
  Extra = 'extra',
}

enum Orientation {
  Horizontal = 'horizontal',
  Vertical = 'vertical',
}
```
