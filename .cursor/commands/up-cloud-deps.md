1. Найди в проекте устаревшие пакеты с помощью команды pnpm outdated:
   ```bash
   pnpm outdated --json
   ```

2. Среди устаревших пакетов, выбери только те, что в названии содержат скопы `@sbercloud/*` или `@snack-uikit/*`.
   Если таких пакетов нет, то останови выполнение команды.

3. Обнови версии пакетов со скопами `@sbercloud/*` и `@snack-uikit/*` до самых актуальных, в файле `package.json`.

4. Запусти команду `pnpm install`.

5. Изучи документацию пакетов `@sbercloud/*` и `@snack-uikit/*`,
   посмотри на контекст их использования и внеси нужные изменения, если найдешь какие-либо BREAKING CHANGES.

6. Проверь, есть ли ошибки линтера, если что исправь:
   ```bash
   npx eslint src --fix
   ```

7. Проверь, если ли ошибки typescript, если что исправь:
   ```bash
   npx tsc --noEmit --emitDeclarationOnly false
   ```

8. Проверь, собирается ли проект, если что исправь ошибки:
   ```bash
   pnpm build
   ```

9. Если ошибок нет, апдейт считай успешным!
